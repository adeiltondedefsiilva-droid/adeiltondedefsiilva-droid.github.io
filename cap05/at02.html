Seção 3 — Glossário

HTML
Linguagem de marcação para estruturar páginas.
Define o conteúdo da página.
CSS
Linguagem para estilizar páginas.
Controla cores e fontes.
JavaScript
Linguagem que adiciona interatividade.
DOM
Representação em árvore do HTML.
API
Interface de comunicação entre sistemas.
HTTP
HTTPS
Protocolo de transferência de dados na web.
URL
Endereço web
Endereço único de um recurso na internet.
DNS
Traduz nomes de domínio em endereços IP.
